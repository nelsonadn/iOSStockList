# iOSStockList

iOS app built with SwiftUI to display a list of market indices and a detail screen for each stock.

## Technology

- SwiftUI
- Swift Concurrency (`async/await`)
- MVVM architecture
- Combine
- URLSession
- RapidAPI
- XCTest

## Features

- Market list with search
- Auto-refresh every 8 seconds
- Stock detail screen
- Loading, empty, and error states
- UI tests and unit tests

## API

- The app uses the Yahoo Finance real-time API through RapidAPI.
